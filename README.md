> "École Supérieure de Génie Informatique"

## Mission
> 5 days
- "Grad school student office events management app".

### Must haves
- MVC (from scratch)
- No Framework
- User auth & CRUD
- Email notifications
- Unit tests

## Start the project

### Install packages

```shell
composer install
```

### Launch server

```shell
composer ss
```

### Launch tests

```shell
composer tests
```

### Create ``/.env.local``

Include this.
```dotenv
DB_USER="username"
DB_PASSWORD="password"
DB_NAME="dbname"
```
